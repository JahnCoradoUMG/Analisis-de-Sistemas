# GymFlow — Sistema de Gestión de Gimnasio

GymFlow es un sistema de software para la **gestión integral de gimnasios**, enfocado en digitalizar la operación diaria de socios, membresías, reservas, control de acceso (check-in) y reportes básicos.

El sistema está diseñado para gimnasios pequeños y medianos que actualmente operan con procesos manuales (cuadernos, planillas, WhatsApp), lo que genera errores, doble reserva y poca visibilidad del uso real del negocio.

## 🎯 Problema que resuelve
- Registro manual de socios y membresías.
- Errores en reservas y sobrecupo en clases.
- Falta de control de asistencia y aforo.
- Dificultad para conocer el estado real de socios activos y uso del gimnasio.

GymFlow centraliza esta información en una plataforma web, permitiendo una operación más ordenada, trazable y eficiente.

## 👥 Usuarios principales
- Socios / Miembros del gimnasio
- Recepción / Atención al cliente
- Instructores / Entrenadores
- Administrador del gimnasio

## 📦 Alcance general (v1)
- Gestión de socios.
- Planes de membresía y vigencia.
- Reservas de clases o franjas horarias.
- Check-in y registro de asistencia.
- Control de cupos.
- Roles y permisos.
- Reportes básicos de uso y asistencia.

## 🚫 Fuera de alcance (v1)
- Facturación electrónica.
- Integraciones contables externas.
- App móvil nativa.
- Pasarela de pagos en línea.
